## installation
install scikit-robot
```bash
sudo apt-get install libspatialindex-dev freeglut3-dev libsuitesparse-dev libblas-dev liblapack-dev
pip install scikit-robot
```
install this
```bash
sudo apt install libeigen3-dev
pip install scikit-build
pip install plainmp
```
